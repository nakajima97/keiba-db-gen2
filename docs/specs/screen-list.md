# 画面一覧

| 画面名 | URL | 説明 |
|---|---|---|
| 収支ダッシュボード | `/dashboard` | 収支記録の確認画面。年を選択するとその年の年間サマリー（合計購入額・合計払い戻し額・プラスマイナス・回収率）と日次収支一覧（購入記録がある日のみ）を表示する。回収率は払い戻し額 ÷ 購入額 × 100。100%以上は緑色、100%未満は赤色で表示。 |
| 馬券登録 | `/tickets/new` | 購入馬券の登録画面。レース情報・券種・買い方・馬番・金額を入力して登録する。登録後は前回レース情報を引き継いで同画面に遷移。 |
| 購入馬券一覧 | `/tickets` | 購入した馬券の一覧画面。日付・レース場・レース番号・券種・買い方・購入金額・払い戻し金額・レース結果リンクを表示。日付降順・レース場降順・レース番号降順でソート。無限スクロール（30件/回）。購入金額は合計金額（単価 × 点数）を表示。払い戻し金額はレース結果入力後に自動計算して表示（未確定時は空欄）。 |
| レース結果入力 | `/races/{uid}/result/new` | JRA公式サイトからコピーした払い戻し情報をペーストしてレース結果を登録する画面。フォーマット不正時はエラーを表示して登録を中断する。 |
| レース結果確認・編集 | `/races/{uid}/result/edit` | 登録済みの払い戻し情報を確認・編集する画面。 |
| レース一覧 | `/races` | 登録済みレースの一覧画面。ナビゲーションメニューからアクセス。日付・開催場所でリアルタイムフィルタリング可能。日付・開催場所・レース番号をテーブル表示。行クリックで詳細画面へ遷移。レース情報入力リンクから入力画面へ遷移。テーブルはレスポンシブ対応（横スクロール）。 |
| レース詳細 | `/races/{uid}` | レースの基本情報（開催日・競馬場・レース番号）と出馬表（枠番・馬番・馬名・騎手名・馬体重）を表示する画面。 |
| レース情報入力 | `/races/new` | JRA公式サイトの出馬表をコピー＆ペーストしてレース情報と出走馬を一括登録する画面。競馬場・レース日・レース番号を選択後、出馬表テキストをペーストして保存する。未登録の競走馬・騎手はサーバーサイドで自動登録する。保存後は同画面に留まり、競馬場・レース日・レース番号を引き継ぐ。 |
