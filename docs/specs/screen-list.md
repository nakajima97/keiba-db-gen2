# 画面一覧

| 画面名 | URL | 説明 |
|---|---|---|
| 馬券登録 | `/tickets/new` | 購入馬券の登録画面。レース情報・券種・買い方・馬番・金額を入力して登録する。登録後は前回レース情報を引き継いで同画面に遷移。 |
| 購入馬券一覧 | `/tickets` | 購入した馬券の一覧画面。日付・レース場・レース番号・券種・買い方・購入金額・払い戻し金額・レース結果リンクを表示。日付降順・レース場・レース番号降順でソート。無限スクロール（30件/回）。払い戻し金額はレース結果入力後に自動計算して表示（未確定時は空欄）。 |
| レース結果入力 | `/races/{uid}/result/new` | JRA公式サイトからコピーした払い戻し情報をペーストしてレース結果を登録する画面。フォーマット不正時はエラーを表示して登録を中断する。 |
| レース結果確認・編集 | `/races/{uid}/result/edit` | 登録済みの払い戻し情報を確認・編集する画面。 |
