# 画面一覧

| 画面名 | URL | 説明 |
|---|---|---|
| 収支ダッシュボード | `/dashboard` | 収支記録の確認画面。年を選択するとその年の年間サマリー（合計購入額・合計払い戻し額・プラスマイナス・回収率）と日次収支一覧（購入記録がある日のみ）を表示する。回収率は払い戻し額 ÷ 購入額 × 100。100%以上は緑色、100%未満は赤色で表示。日次収支一覧は日付降順でソート。 |
| 馬券登録 | `/tickets/new` | 購入馬券の登録画面。レース情報・券種・買い方・馬番・金額を入力して登録する。登録後は前回レース情報を引き継いで同画面に遷移。 |
| 購入馬券一覧 | `/tickets` | 購入した馬券の一覧画面。日付・レース場・レース番号・券種・買い方・購入金額・払い戻し金額・レース結果リンクを表示。日付降順・レース場降順・レース番号降順でソート。無限スクロール（30件/回）。購入金額は合計金額（単価 × 点数）を表示。払い戻し金額はレース結果入力後に自動計算して表示（未確定時は空欄）。 |
| レース結果入力 | `/races/{uid}/result/new` | JRA公式サイトからコピーした払い戻し情報をペーストしてレース結果を登録する画面。フォーマット不正時はエラーを表示して登録を中断する。 |
| レース結果確認・編集 | `/races/{uid}/result/edit` | 登録済みの払い戻し情報を確認・編集する画面。着順テーブルに「メモ」列を持ち、メモが存在する馬は内容（プレーンテキスト 1000字以内）を表示する。表示優先順位は (1) そのレースに紐づくメモ → (2) レースに紐づかないメモ → (3) どちらもなければ追加ボタン表示。メモ列のクリックで追加・編集モーダルを起動する。 |
| レース一覧 | `/races` | 登録済みレースの一覧画面。ナビゲーションメニューからアクセス。日付・開催場所でリアルタイムフィルタリング可能。日付・開催場所・レース番号・レース名をテーブル表示（レース名はnullableのため未登録時は「—」を表示）。行クリックで詳細画面へ遷移。レース情報入力リンクから入力画面へ遷移。テーブルはレスポンシブ対応（横スクロール）。 |
| レース詳細 | `/races/{uid}` | レースの基本情報（開催日・競馬場・レース番号・レース名）と出馬表（枠番・馬番・馬名・騎手名・馬体重）を表示する画面。レース名はnullableのため未登録時は「—」を表示。出馬表に「自分の印」列（1列固定）と「他人の印」列（0列以上、ラベル自由入力・追加・削除可能、当該レース内のみ管理）を持ち、各セルはセレクトボックスから6種類の印（◎・○・▲・△・×・✓）を選択できる。印の変更は自動保存。出走馬登録画面へのリンクを持つ。馬名の右にメモアイコンを表示し（メモあり=ノートアイコン／メモなし=＋アイコン）、クリックでメモ追加・編集モーダルを起動する。メモあり判定の優先順位は (1) そのレースに紐づくメモ → (2) レースに紐づかないメモ → (3) どちらもなければアイコンは「メモなし」状態。他人の印列（type=other）の各印セル内には印メモアイコン（ノート風アイコン）を表示し、クリックで印メモ追加・編集モーダルを起動する。印メモアイコンは「印あり」または「印メモあり」の場合のみ表示し、両方ともない場合は非表示（画面のごちゃつき防止）。印を変更・解除しても印メモは残る。自分の印列（type=own）には印メモ機能はない。 |
| 出走馬登録 | `/races/{uid}/entries/new` | レース詳細から遷移して出走馬を登録する画面。JRAの出馬表テキストをペーストして登録する。登録後はレース詳細画面へ遷移。 |
| レース情報入力 | `/races/new` | JRA公式サイトの出馬表をコピー＆ペーストしてレース情報と出走馬を一括登録する画面。競馬場・レース日・レース番号を選択後、任意でレース名を入力し、出馬表テキストをペーストして保存する。未登録の競走馬・騎手はサーバーサイドで自動登録する。保存後は同画面に留まり、競馬場・レース日・レース番号を引き継ぐ。 |
| 競走馬詳細 | `/horses/{horse}` | 競走馬の基本情報（馬名・生年）と、その馬のレース履歴一覧（開催日・競馬場・レース番号・レース名・着順・騎手・人気）を表示する画面。レース詳細画面・レース結果確認画面の馬名リンクから遷移。同画面にメモ一覧（レース紐づきあり・なし含む全メモ）と「メモを追加」ボタンを表示し、ボタン押下でメモ追加モーダル（任意でレース指定可）を起動する。 |
